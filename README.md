# MailAnalysis
Japanese Only

以前業務で製作した物(うろ覚え)を、ポートフォリオ用に1から作り直し中  


## このPHP classの説明
受信メールの解析  
メールのソースを渡し、送受信アドレス、件名、本文、添付ファイルなどの解析、抽出する

### 業務で使用した内容
- 空メール登録
- メール送信後のエラーメール処理
- ブログのメール投稿処理

### 動作テスト用URL
- https://8ware.site/mailanalysis/index.php

### 業務で制作した際にあった機能で追加予定のない機能
- エラーメールのエラー内容の判定

### できあがっている部分
- ヘッダ部分の抽出（一部 To,From など）
